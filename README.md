# Even or Odd Checker (Java)

This is a simple Java console-based program.  
The program takes a number from the user and checks whether the number is even or odd.

This project is created for learning Java basics and practicing simple logic using if-else statements.

## Program Explanation
- The program asks the user to enter a number.
- It checks the number using the modulus (%) operator.
- If the number is divisible by 2, it is even.
- Otherwise, the number is odd.
- The result is displayed on the console.

## Technologies Used
- Java
- Scanner class

## How to Run the Program
1. Open terminal or command prompt in the project folder.
2. Compile the program using:
   javac EvenOdd.java
3. Run the program using:
   java EvenOdd
