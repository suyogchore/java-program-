# Simple Calculator (Java)    

Basic Java console application for arithmetic operations.     
  
A simple console-based calculator program written in Java.    
                                                         
## Features                                                                                                                                       
- Addition          
- Subtraction          
- Multiplication    
- Division       
- Beginner-friendly code          
 
## Technologies Used    
- Java   
- Scanner Class   
   
## How to Run  
1. Compile the program:
   javac Calculator.java
2. Run the program:
   java Calculator
